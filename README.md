# openvino.skills
Some skills for openvino development


# How to use
1. Clone the repository
```bash
git clone https://github.com/openvinotoolkit/openvino.skills.git
```

2. Copy the skills to your local repository
```bash
cp -r openvino.skills/gpu_plugin/* <your_local_repository>/.github/skills/.
```

3. Use the skills in your workflow
